# FirstBrief corporate UI

The application uses an original operational interface aligned to the public
NATS digital language observed in July 2026:

- deep navy foundation `#071920`;
- corporate blue `#145569`;
- cyan accent `#46c3d7`;
- green success accent `#50c364`;
- strong headings, generous white space and crisp rectangular controls.

The FirstBrief airspace-line mark is original and is not the NATS logo. No NATS
website content or proprietary font files are bundled. System fallbacks provide
Roboto/Montserrat-like proportions without external tracking.

All states retain text labels, focus uses a high-visibility cyan ring, responsive
layouts do not hide authorised functions, and reduced-motion preferences are
respected. Corporate brand-owner approval remains required before production.
